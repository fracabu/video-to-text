<a name="english"></a>
## :gb: English

### Overview

An application to convert videos into text by extracting audio, transcribing it, and translating the text into multiple languages. Perfect for accessibility, subtitles, and global reach.

### Features

- **Audio Extraction** - Extract audio track from video files
- **Transcription** - Convert speech to text automatically
- **Multi-Language Translation** - Translate to multiple languages
- **Subtitle Generation** - Create SRT/VTT subtitle files
- **Batch Processing** - Process multiple videos

### Use Cases

| Use Case | Description |
|----------|-------------|
| Accessibility | Make videos accessible to deaf/hard of hearing |
| Subtitles | Generate subtitles for video content |
| Translation | Reach global audiences |
| Content Repurposing | Convert video to text articles |

### Quick Start

```bash
git clone https://github.com/fracabu/video-to-text.git
cd video-to-text

npm install

npm start
```

### Requirements

- Node.js 18+
- FFmpeg installed

## :it: Italiano

### Panoramica

Un'applicazione per convertire video in testo estraendo l'audio, trascrivendolo e traducendo il testo in piu lingue. Perfetto per accessibilita, sottotitoli e reach globale.

### Funzionalita

- **Estrazione Audio** - Estrai traccia audio dai file video
- **Trascrizione** - Converti parlato in testo automaticamente
- **Traduzione Multi-Lingua** - Traduci in multiple lingue
- **Generazione Sottotitoli** - Crea file sottotitoli SRT/VTT
- **Elaborazione Batch** - Processa piu video

### Casi d'Uso

| Caso d'Uso | Descrizione |
|------------|-------------|
| Accessibilita | Rendi video accessibili a sordi/ipoudenti |
| Sottotitoli | Genera sottotitoli per contenuti video |
| Traduzione | Raggiungi audience globali |
| Repurposing Contenuti | Converti video in articoli testuali |

### Avvio Rapido

```bash
git clone https://github.com/fracabu/video-to-text.git
cd video-to-text

npm install

npm start
```

### Requisiti

- Node.js 18+
- FFmpeg installato
